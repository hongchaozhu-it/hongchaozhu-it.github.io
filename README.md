这是一个改造成中文内容的 Jekyll GitBook 风格示例站点。它适合放博客、知识笔记、项目文档，也适合用来搭建个人手册。

## 这个主题现在有什么

- 侧边栏目录清晰，适合持续写作和归档文章。
- 首页、文章页、独立页面都已经换成中文示例内容。
- 支持目录、封面图、宽表格、提示块、Mermaid、数学公式和脚注。
- 文章使用 Markdown 编写，适合在 Git 仓库里长期维护。

## 阅读建议

如果你是第一次接触这个主题，可以按下面的顺序浏览：

1. 从《为什么选择 Jekyll 与 GitBook 风格》了解整体思路。
2. 再看《如何开始搭建中文知识站》，熟悉基本配置。
3. 最后浏览后面的示例文章，看看目录、封面图、图表和公式等效果。

## 适合放什么内容

- 技术博客
- 学习笔记
- 产品文档
- 内部知识库
- 个人写作手册

## 中文示例内容说明

这份示例保留了原主题的大部分功能展示，但把首页、文章和静态页面改成了中文写法，方便你直接继续替换为自己的内容，而不用先清理整站英文示例。

> ##### 提示
>
> 如果你准备把这里改成自己的站点，最先需要调整的一般是 `_config.yml`、`README.md`、`_posts` 和 `_pages` 目录。
{: .block-tip }

## 接下来可以继续改什么

- 修改站点标题、描述与作者信息
- 替换示例文章为自己的内容
- 调整导航顺序和分类
- 增加自定义样式或统计脚本
- 部署到 GitHub Pages
